## 2. Load the Iris Dataset

The Iris dataset is one of the most popular datasets in machine learning.

It contains measurements of iris flowers from three species:

- Setosa
- Versicolor
- Virginica

The dataset includes four features:

1. Sepal Length
2. Sepal Width
3. Petal Length
4. Petal Width
